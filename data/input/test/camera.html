相对于 Main 视角的多相机方案

所有相机在半径 10 的球面上。上下沿 main 的 y_view 所在大圆偏转，左右沿 main 的 x_view 所在大圆偏转。

Main 相机
Up / Down 圆周
Left / Right 圆周

红色/绿色/蓝色短箭头分别表示每个相机的 x_view、y_view、z_view。

up = cos(45°) * main_z + sin(45°) * main_y
down = cos(45°) * main_z - sin(45°) * main_y
right = cos(45°) * main_z + sin(45°) * main_x
left = cos(45°) * main_z - sin(45°) * main_x

拖拽旋转，滚轮缩放。彩色小弧线标出从 main 到各侧视角的 45° 旋转角。
